A quick reference for common Markdown syntax.

## Headings

```markdown
# H1
## H2
### H3
```

## Lists

```markdown
- Unordered item 1
- Unordered item 2

1. Ordered item 1
2. Ordered item 2
```

## Code Blocks

````markdown
```python
print("Hello")
```
````

## Links & Images

```markdown
[Link text](https://example.com)
![Image alt](image.png)
```

## Blockquotes

```markdown
> Quoted content
```
