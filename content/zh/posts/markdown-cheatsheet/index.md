记录一些常用的 Markdown 语法，方便随时查阅。

## 标题

```markdown
# H1
## H2
### H3
```

## 列表

```markdown
- 无序项 1
- 无序项 2

1. 有序项 1
2. 有序项 2
```

## 代码块

````markdown
```python
print("Hello")
```
````

## 链接与图片

```markdown
[链接文字](https://example.com)
![图片描述](image.png)
```

## 引用

```markdown
> 引用内容
```
